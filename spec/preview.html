Screen Flow & Navigation

The app has a simple two-screen structure. The face screen is always the primary view. Settings are accessed via a hidden triple-tap gesture on the top-right corner and presented as a modal sheet. In production (Guided Access / Single App Mode), the settings gesture is disabled at OS level.

Automatic Emotion State Machine

  System State Check (Priority Order)
  ====================================

  networkStatus == .disconnected  ---->  .error     (Red X eyes)
          |
          v (connected)
  batteryLevel < 0.1             ---->  .lowBattery (Orange droopy)
          |
          v (battery OK)
  isCharging == true             ---->  .charging   (Light blue relaxed)
          |
          v (not charging)
  streamingStatus == .streaming  ---->  .happy      (Cyan smile)
          |
          v (not streaming)
  streamingStatus == .error      ---->  .error      (Red X eyes)
          |
          v (idle)
  default                        ---->  .normal     (White round)

  Additional Triggers:
  - RTSP client connects  --->  .wink (momentary)  --->  .happy
  - Cleaning mode active  --->  .cleaning (Green sparkle)
